![Screenshot of map inspector](./screenshot.png "Map inspector with Surreal loaded")

## Development

Install dependencies:

```console
npm install
```

Run the dev server:

```console
npm start
```

### Running scripts

[tsx](https://tsx.is) is included to run TypeScript files directly.

Example:

```console
tsx scripts/generate-manifest.ts
```
