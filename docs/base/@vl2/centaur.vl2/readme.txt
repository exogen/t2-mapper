* LEVEL DESCRIPTION *
--------------------------------------------------------------------------------
Centaur is a siege level on a lush mountain island. The Offensive team has
access to a vehical pad which will be one key to taking the island. The
defensive team must use their height to an advantage. Maintaining the AA Support
will be a vital role in defending the base. 

Bots: The bot support is a little odd. For some reason, the map may crash if
you enable bot support. To work around this issue. Load the map without bots
enabled and then reload yet again, this time with bots enabled. 

If any one knows the cause of this error please fill free to contact me.
